Time series data

For all energy components the capacity or operation rate can be provided as time series for each region.

This data can be provided as excel files for easy access. An example can be found here.

The following attributes are available for maximum capacities:

pydantic model ensysmod.schemas.CapacityMaxCreate[source]

Attributes to receive via API on creation of a max capacity.

field component: str [Required]

The name of the component.

Constraints

  • example = heat_pump

field max_capacities: List[float] [Required]

Max capacities for a component in a specific region. Provide single value or a list of values for each time step in dataset.

Constraints

  • example = [1.0, 2.0, 3.0]

field ref_dataset: int [Required]

The ID of the referenced dataset. Current dataset is used as default.

field region: str [Required]

The name of the region.

Constraints

  • example = germany

field region_to: Optional[str] = None

Optional region to name, if needed.

Constraints

  • example = france
