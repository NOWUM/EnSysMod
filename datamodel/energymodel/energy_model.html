Energy model

The energy model is used to create a simulation based on a dataset. The model parameters allows to override the default values provided by the dataset.

pydantic model ensysmod.schemas.EnergyModelCreate[source]

Attributes to receive via API on creation of an energy model.

field description: Optional[str] = None

Description of the energy model

Constraints

  • example = A model that reduces CO2 emissions by 100%

field name: str [Required]

Name of the energy model.

Constraints

  • example = 100% CO2 reduction

field parameters: Optional[List[ensysmod.schemas.energy_model_parameter.EnergyModelParameterCreate]] = None

Parameters of the energy model. The parameters override the values of the referenced dataset.

Constraints

  • examples = [EnergyModelParameterCreate(ref_dataset=None, component=’CO2 to environment’, region=None, region_to=None, attribute=<EnergyModelParameterAttribute.yearly_limit: ‘yearly_limit’>, operation=<EnergyModelParameterOperation.set: ‘set’>, value=0.0, ref_model=None)]

field ref_dataset: int [Required]

ID of the dataset that the energy model is based on.

Constraints

  • example = 1
